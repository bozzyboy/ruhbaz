# User Profiles

Bu klasör yalnızca kullaniciya ait profiller ve kullanıcının oluşturduğu diğer insan kimlikleri icindir.

## Onerilen Yapi

- `users/<user-id>/profile.md`
- `users/<user-id>/preferences.json`
- `users/<user-id>/personas/<persona-id>.md`

Assistant personelar bu klasore konmaz.
Falcilar ve diğer uygulama karakterleri `../assistants/` altinda tutulur.
